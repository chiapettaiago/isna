# Instruções para Adicionar Logos dos Parceiros do Bairro

Para que a nova seção "Parceiros do Bairro" funcione corretamente, você precisa adicionar as seguintes imagens no diretório `/home/iago/projects/isna/images/parceiros/`:

## Imagens Necessárias:

1. **af-fitness.jpg** - Logo do AF Fitness (Centro de Treinamento)
2. **elvis-hernandez.jpg** - Logo do Centro de Treinamento Elvis Hernandez
3. **alem-do-disfarce.jpg** - Logo da Barbearia Além do Disfarce
4. **mercadinho-felpao.jpg** - Logo do Mercadinho Felpão

## Como Adicionar:

1. Extraia/recorte os logos da imagem fornecida (WhatsApp Image 2025-10-30 at 22.46.00.jpeg)
2. Salve cada logo individualmente com os nomes acima
3. Formato recomendado: JPG ou PNG (fundo transparente se possível)
4. Tamanho recomendado: mínimo 300x300px, máximo 800x800px
5. Copie os arquivos para: `/home/iago/projects/isna/images/parceiros/`

## Comandos Úteis:

```bash
# Ver imagens atuais no diretório
ls -lh /home/iago/projects/isna/images/parceiros/

# Copiar imagens (exemplo)
cp /caminho/para/suas/imagens/*.jpg /home/iago/projects/isna/images/parceiros/
```

## Verificação:

Após adicionar as imagens, acesse a página principal do site e verifique se a seção "Parceiros do Bairro" está exibindo todos os logos corretamente.
